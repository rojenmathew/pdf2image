# PDF TO IMAGE TOOL 

![](https://github.com/parikshitgupta1/pdf2image/blob/master/screen.png?raw=true)

## Set up environment
### Create virtual environment:
    python -m venv venv
### Activate virtual environment (windows):
    venv\Scripts\activate.bat
    
## How to run
### Install environment:
   Run: install.bat
### Run app:
   Run: main.bat
